Tha Basic Language of the Web: HTML

Posted by Laura Jones on Monday June 21st 2017

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and JavaScript. Tese are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for HyperText Markup Language. It's markup language that web developers use to structure and describe the contect of a webpage (not a programming language).

HTML consists of elements that describe different types of content: paragraphs, links. headings. images, video, etc. Web browsers understand HTML and render HTML code as webistes.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs.
